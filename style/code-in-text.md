# Code in text

*   **Use code font for:** Classes, methods, commands (`gcloud`), filenames, paths, HTTP codes (`404`), SQL keywords, parameters.
*   **Do not use code font for:** Product names, domain names (unless in code), URLs (use links).
*   **UI Elements:** If a UI element represents code (like a filename), use bold code font (**`filename.txt`**).
*   **Plurals/Possessives:** Do not apply grammar to code terms. Rewrite to avoid (e.g. "Value of `wordCount`" not "`wordCount`'s value").
