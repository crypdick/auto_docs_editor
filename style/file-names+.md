# Filenames and file types

*   **Names:** Lowercase with hyphens (`query-data.html`). No underscores (unless matching existing pattern).
*   **Referencing:** Use code font + "file" (the `build.sh` file).
*   **Types:** Use formal names (PNG file), not extensions (`.png` file), unless discussing the extension itself.

Do not apply this rule to real URLs or links; this will break the link!
