# Command-line terminology

* gcloud:** Use `command`, `group`, `flag`, and `argument`.
* Linux:** Use `command`, `option` (starts with `-`), and `argument`.
* General:** *Option* is a safe catchall term.
* Signals:** Use specific terms:
* `KILL`: *kill*
* `TERM`: *terminate*
* `QUIT`: *quit*
* `INT`: *interrupt*
* `PAUSE`: *pause*
