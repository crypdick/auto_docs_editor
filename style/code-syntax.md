# Command-line syntax

* **Formatting:** Use code blocks. Indent 4 spaces.
* **Line continuation:** Use `\` (Linux) or `^` (Windows).
* **Prompts:** Optional (`$`). Use separate code blocks for input and output.
* **Syntax:**
  * `[optional]`
  * `{exclusive|choice}`
  * `...` (repeating)
* **Click-to-copy:**
  * Do not use abstract syntax (`[]`, `{}`, `...`) in copyable examples.
  * Use concrete values or placeholders.
  * Separate input commands from output.
