# Product names

*   **Capitalization:** Use Title Case for official product names (*Google Cloud*). Follow official branding.
*   **Features:** Use lowercase for generic feature names unless officially capitalized.
*   **Articles:**
    *   **No 'the':** Before product names (*Cloud Datastore*).
    *   **Use 'the':** Before tool/API names (*The Transcoder API*, *The gcloud CLI*).
*   **Verbs:** Do not use product names as verbs.
